024 * <h2>Overview</h2>
025 * Builder for creating {@link java.util.Collection}s providing some convenience methods. The class
026 * writes
027 * everything through into the contained collector. Using the default constructor a newly created
028 * {@link java.util.ArrayList} will be used as collector, but you can pass you own collector as
029 * constructor-argument. Of course this should be mutable in order to work.
030 * <h3>Handling of null-values</h3>
031 * <p>
032 * As default {@code null} values are ignored. This behavior can be changed by call
033 * {@link #addNullValues(boolean)}.
034 * <em>Caution:</em> In case of using one of the {@link #copyFrom(Collection)}
035 * methods for instantiation the {@code null} values will not be checked in that way
036 * </p>
037 * <h3>Standard Usage</h3>
038 *
039 * <pre>
040 *
041 * List&lt;String&gt; result = new CollectionBuilder&lt;String&gt;().add("this").add("that")
042 *         .add(mutableList("on", "or an other")).toImmutableList();
043 * </pre>
044 *
045 * or
046 *
047 * <pre>
048 *
049 * Set&lt;String&gt; result = new CollectionBuilder&lt;String&gt;().add("this").add("that")
050 *         .add(mutableList("on", "or an other")).toMutableSet();
051 * </pre>
052 *
053 * <h3>Copy From</h3>
